Этот каталог предназначен для Laravel-проекта.

В среде помощника Composer недоступен, поэтому скелет Laravel не установлен автоматически.
На своей машине выполните:

composer create-project laravel/laravel backend

После этого перенесите/совместите файлы из каталога backend/app, backend/routes и т.д.,
которые создаст помощник, с вашим установленным проектом.
